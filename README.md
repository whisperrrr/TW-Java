##需求描述**
* Student类中有姓名和年龄两个属性
* Teacher类中有姓名和学生两个属性，学生的类型为Map，按年龄分类，有addStudent方法
* Application中，有Teacher，姓名Bob，学生(Linda，18), (Cindy, 19), (Mary, 19)
* 给Bob添加学生(Ben, 19),（Solider, 20）
* 打印所有学生的信息
